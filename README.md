# VoIP Platform Documentation

This repository contains comprehensive documentation for the VoIP platform.

## Directory Structure

- `docs/markdown/` - Markdown documentation files
- `docs/diagrams/` - System architecture diagrams (Mermaid)
- `docs/planning-steps/` - Development planning documents

## Documentation

### System Architecture
See [SYSTEM_ARCHITECTURE.md](docs/markdown/SYSTEM_ARCHITECTURE.md) for complete technical architecture.

### Diagrams
See [docs/diagrams/](docs/diagrams/) for visual system diagrams including:
- High-level architecture
- VoIP call flows
- SMS/MMS workflows
- Authentication flows
- Database schema
- And more...

## Getting Started

1. Review the system architecture document
2. Explore the diagrams for visual understanding
3. Check planning steps for development phases

## Viewing Diagrams

All diagrams use Mermaid syntax and can be viewed:
- Natively on GitHub
- In VS Code with Mermaid extension
- On [Mermaid Live Editor](https://mermaid.live)
